## Project Overview

This project converts Context-Free Grammars (CFG) to Pushdown Automata (PDA) and vice versa, with visualization and step-by-step explanation.

## Features

* CFG → PDA conversion
* PDA → CFG conversion
* String testing (CYK-based)
* Visual diagrams and walkthrough
